### 371. 两整数之和


##### 1.题目
给你两个整数 a 和 b ，不使用 运算符 + 和 - ​​​​​​​，计算并返回两整数之和。

##### 2.示例
示例1:
```
输入：a = 1, b = 2
输出：3
```

示例2:
```
输入：a = 2, b = 3
输出：5
```

##### 3.思路
- 思路不太清楚

##### 4.代码
```javascript
var add = function(a, b) {
  while (b != 0) {
      const carry = (a & b) << 1;
      a = a ^ b;
      b = carry;
  }
  return a;
};

// loop1
// 0111111  a
// 0001111  b
// 0001111  a & b
// 0011110  (a & b) << 1  carry 
// 0110000  a = a^ b
// 0011110  b = carry

// loop2
// 0110000  a
// 0011110  b
// 0010000  a & b
// 0100000  (a & b) << 1  carry 
// 0101110  a = a^ b
// 0100000  b = carry

// loop3
// 0101110  a
// 0100000  b
// 0100000  a & b
// 1000000  (a & b) << 1  carry 
// 0001110  a = a^ b
// 1000000  b = carry

// loop4
// 0001110  a
// 1000000  b
// 0000000  a & b
// 0000000  (a & b) << 1  carry 
// 1001110  a = a^ b
// 0000000  b = carry

// result a => 1001110 => 78
console.log(add(63, 15)) // 78
```
